## Overview

Path Finder Professional Suite is a complete file management platform that serves power users and system administrators with the browsing depth, operational breadth, and customization flexibility needed for professional file system management. It recognizes that serious file management requires tools beyond what consumer-oriented file browsers provide, offering the depth that technical users need without requiring everything to happen at the command line. The modular interface adapts to any file management workflow through customizable shelf panels.

Module shelf displays configurable panels including disk usage visualization, file preview, hex inspector, metadata viewer, Dropbox status, and Git repository status alongside the main file browser, creating a customized file management dashboard tailored to your specific workflow. Modules can be rearranged, resized, and toggled on and off depending on the current task. Custom module configurations can be saved as workspace presets for different file management contexts.

Smart folders and saved searches create dynamic file collections based on search criteria, providing instant access to files matching specific conditions regardless of their physical location in the file system. Search criteria support Spotlight metadata attributes, enabling searches based on content, camera model, document author, and any other indexed attribute. Saved searches update dynamically and can be bookmarked in the sidebar for persistent access.

Bookmark and drop stack features provide quick access to frequently used directories and a staging area for collecting files from multiple locations before performing batch operations on the collected set. The drop stack accumulates files from any location and any number of browsing sessions, then applies batch operations — move, copy, compress, rename — to the entire collected set at once. Bookmarks support keyboard shortcuts for instant navigation to deep directory paths.

Path Finder Professional Suite is the file management platform for users who spend significant time navigating, organizing, and manipulating files and want a tool that matches the sophistication of their file management needs. Its depth of capability and decades of refinement make it the advanced file manager that power users trust for serious file system work.

## Performance & Integration

Path Finder is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Path Finder processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
